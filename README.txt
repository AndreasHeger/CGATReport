SphinxReport 
=============

The SphinxReport module is an extension for sphinx
that provides facilities for data retrieval and data rendering
within reStructured text. 

To install system-wide on Unix-like systems:

sudo python setup.py install

or (for system without sudo):

su -c python setup.py install


Documentation
=============

See the `homepage <http://wwwfgu.anat.ox.ac.uk/~andreas/SphinxReport/index.html>` for
documentation and tutorials.
